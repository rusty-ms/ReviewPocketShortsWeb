## Overview

This is the companion website for the **ReviewPocketShorts** YouTube Shorts channel. It hosts a product page for every video posted, with affiliate links, product details, and an embedded YouTube Short.

**Live site:** https://rusty-ms.github.io/ReviewPocketShortsWeb/  
**Custom domain:** (configure via CNAME if desired)

---

## How It Works

This repo is **auto-updated by the pipeline** — you never need to push to it manually.

Every time a new video is generated by the ReviewPocketShorts pipeline on `vps-n8n`:

1. `website_publisher.py` clones this repo fresh
2. Generates a new product page at `/products/{ASIN}.html`
3. Updates `products.json` (product index + featured product)
4. Regenerates `sitemap.xml`
5. Commits and pushes — GitHub Pages deploys automatically within ~30 seconds

## Manual Operations

### Force a re-publish from the pipeline server
```bash
ssh vps-n8n "cd /opt/ReviewPocketShorts && source venv/bin/activate && python -c \"
from scripts.website_publisher import publish_to_website
# publish_to_website(product, yt_result) — fill in product/yt dicts
\""
```

### Check deployment status
GitHub Pages deploys within ~30 seconds of a push. Check:
- https://github.com/rusty-ms/ReviewPocketShortsWeb/actions
- Or just visit the live URL
